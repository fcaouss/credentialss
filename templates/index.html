What it does

Everything. Configured your way.

Every setting is per-server, configurable from the web dashboard or bot commands. Nothing locked behind a paywall.

Full Moderation Suite

Ban, kick, timeout, mute, warn, purge, lock channels, set slowmode, and manage nicknames. Every action gets logged to your designated mod log channel.

bankicktimeoutwarnmutepurgelock

Web Dashboard

Log in with Discord. See all your servers. Configure everything — roles, permissions, verification, tickets — without touching a command.

Role-Based Permissions

Set which roles are required to use any command. Mark roles as bypass so they skip all restrictions. Per-command, per-server.

Giveaway System

Start timed giveaways with multiple winners. Enable rigging to guarantee a specific user wins while still looking fair to everyone else. Reroll anytime.

giveaway start--rig @usergiveaway endgiveaway reroll

Verification System

Auto-create an Unverified role, verification channel, and button. Account age requirements, bot kick-on-join, and full logging.

Ticket System

Deploy a ticket panel with one command. Private support channels created instantly. Add/remove members. Close cleanly.

Welcome & Leave Messages

Custom messages with variables for username, server name, and member count. Auto-assign a role on join.

Fun Commands

8ball, coinflip, dice rolls, RPS, roast, compliment, ship, rate, mock, jokes, and facts. Keeps things active.

Getting Started

Up in under three minutes.

01

Invite the bot

Click the invite link above and select your server. Grant Administrator for full functionality.

02

Open the dashboard

Log in with Discord at the top of this page. Your server will appear in the dashboard. Configure everything from there.

03

Set your prefix (optional)

Run /prefix ! to change the command prefix. All commands work with the new prefix immediately.

04

Set up verification (optional)

Run /verification setup to auto-create the Unverified role and verification channel in one shot.
